## Getting Started

```bash
# Clone
git clone https://github.com/nodebase/nodebase.git
cd nodebase

# Install
npm install

# Setup environment
cp .env.example .env.local

# Run
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
