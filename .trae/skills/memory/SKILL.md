# Memory

This skill allows for storing and retrieving persistent information across sessions.

## Usage

Use this skill when you need to:
- Remember user preferences or project details
- Recall previously stored information
- Manage a knowledge base for the project
- Store facts that should be available later

## How it works

The skill provides tools to read from and write to a persistent memory store.
