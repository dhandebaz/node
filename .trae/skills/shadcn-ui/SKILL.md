# shadcn/ui

This skill integrates with shadcn/ui to help manage UI components.

## Usage

Use this skill when you need to:
- Add new shadcn/ui components to the project
- Configure existing components
- Get information about available components
- Generate component code

## How it works

The skill uses the `shadcn` CLI via MCP to interact with the library.
