# Codebase Summarizer

Get a bird's eye view of the project and its core architecture.

## Summary Template

### 🏗 Architecture
- Core design patterns
- Major technology stack
- Project layout overview

### 🔑 Key Modules
- **Authentication**: How users log in and out
- **Multi-tenancy**: How data is isolated
- **Business Logic**: Core services and domain models
- **Database**: Schema management and RLS
- **API**: Endpoint structure and versioning

### 🛠 Tech Stack
- Frontend framework/library
- Backend framework/runtime
- Database/ORM
- Testing framework
- CI/CD tools

### 📦 Project Structure
- `src/app`: Next.js pages/routes
- `src/components`: UI components
- `src/lib`: Core services and utilities
- `migrations`: Database schema updates

## Best Practices

- Keep summary up-to-date
- Focus on high-level architecture
- Highlight critical modules
- Document major technical debt
- Provide entry points for new developers
- Link to more detailed documentation
